
1 При установке приложения запрашивается разрешение на определение местоположения.
Если вы даёте разрешение, приложение при его открытии каждый раз автоматически определяет 
ваше местоположение и предоставляет погодные условия для вашего текущего местоположения.
При повороте устройства приложение приложение также переключается на показ погоды
для текущего местоположения
Если вы по каким либо причинам разрешение не даёте, либо устройство не смогло определить
местоположение, приложение предоставляет погодные условия для города,
который был выбран последним в прошлый раз

2 Для погоды отображаются погодное состояние, температура, скорость ветра, давление.
Для прогноза погоды на 5 дней отображаются температура и осадки.
При этом показ/скрытие данных по скорости ветра и давлению можно установить в настройках 
Если ваше устройство имеет встроенные датчики температуры и влажности, через настройки можно
установить вывод температуры и влажности в месте расположения устройства.

3 Список городов, для которых выводятся погодные данные, вы заполняете самостоятельно. 
Для изменения города нажмите круглую кнопку внизу экрана (или на текст "Показать город"
в альбомном положении экрана).
Название российского города в появившемся диалоговом окне
можно вводить как в русском, так и  в латинском регистре, например для ввода города Москва
можно ввести Москва, москва, Moscow или moscow.
Название города мира нужно вводить буквами в латинском регистре, например для ввода города Берлин
нужно ввести Berlin или berlin.
Если существует несколько городов с одноимённым названием, следует добавить к названию города
через запятую двухбуквенный код страны.
Например, если требуется город Берлин не в Германии, а в США, можно ввести berlin,us.
Коды стран можно посмотреть здесь https://ru.wikipedia.org/wiki/ISO_3166-1.
Написание некоторых городов мира по-английски - здесь
https://www.homeenglish.ru/Othercityworld.htm

4 Города можно добавлять в список через пункт меню "Добавить город".Если впоследствии -
 при щелчке на данном городе - окажется, что такой город не существует,
 выведется всплывающее сообщение и город будет удалён из списка.

5 Удалять города из списка можно через контекстное меню списка "Удалить город". 
Контекстное меню вызывается при долгом нажатии на строке списка. 
Очистить список можно через пункт контекстного меню "Очистить список".

6 Погодные данные городов из списка сохраняютс в базе данных.  И если с момента повторного 
обращения за погодными данными того же города прошло менее часа, данные читаются из базы данных,
в противном случае посылается  запрос на погодный сервер.
